# Brand assets

Drop the **official** HDFC Bank logo here. It is not committed to this repo: the mark is a
registered trademark, and a hand-drawn approximation would be both legally awkward and visibly
wrong next to the real one.

| File | Where it appears | Notes |
|---|---|---|
| `hdfc-logo.*` | Expanded sidebar (full wordmark) | **SVG, PNG, JPG or WEBP all work** — the component tries each extension in turn. PNG should be ≥2x (about 260×64) or it will look soft on a Retina display. |
| `hdfc-mark.*` | Collapsed sidebar (square mark only) | Optional. Without it, the collapsed rail falls back to a monogram tile. |

If neither file is present the header falls back to a neutral "H" tile. Nothing breaks.

The logo renders on a white plate in both light and dark themes. That is deliberate: the mark is
designed for a white field, and recolouring or reversing a trademark is normally a brand-guideline
violation. Giving it the background it was designed for is not.
